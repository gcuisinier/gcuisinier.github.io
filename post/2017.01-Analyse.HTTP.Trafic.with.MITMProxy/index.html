Analysing HTTP trafic with MITMProxy

tldr;

If you need to sniff HTTP trafic, first run the MITMProxy (and his companion MITMWeb). Thanks to Docker, this is quite simple

docker run --rm -d --name mitmproxy -p 8080:8080 -p 127.0.0.1:8081:8081 mitmproxy/mitmproxy mitmweb --web-iface 0.0.0.0

Then, configure your software to use the proxy on localhost:8080 ( or the docker-machine IP).
