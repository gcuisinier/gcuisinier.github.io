jEnv

A simple way to switch JAVA_HOME per directory

When you work with multiple JDK version, you probably have to change JAVA_HOME frequently. jEnv is a solution to help you !

It allows you to configure which version to use for a specific folder, and jEnv will change JAVA_HOME each time you change of directory.
